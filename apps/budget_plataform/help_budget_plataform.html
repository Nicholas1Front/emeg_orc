Ajuda com a criação de orçamento

Bem-vindo à página de ajuda!

Aqui você encontrará um guia completo para utilizar o sistema de criação de orçamento de forma correta e eficiente.

Do que se trata este sistema ?

O sistema de criação de orçamento é uma plataforma web focada na criação e administração de orçamentos da EMEG. Este sistema funciona localmente porém é necessária conexão com internet para acessar as informações e dados dos clientes que estão na nuvem.

Em alguns casos pode ser necessária a adição de algum cliente ou edição de um equipamento, para isso o sistema de controle de clientes estará disponível e funcional para realizar essas alterações.

Dados para preenchimento

"Dados para preenchimento" é o cabeçalho do orçamento, o que significa dizer que as informações que os campos exigem são necessárias e importantes para identificar o orçamento e tornar o mesmo mais detalhado e preciso tanto para EMEG quanto para o cliente que solicitou o orçamento em questão.

Após todas as informações serem devidamente colocadas em seus respectivos campos o botão "Próximo passo" deve ser clicado para que o sistema possa liberar com segurança as outras áreas de preenchimento do orçamento.

Abaixo seguem algumas informações a respeito do preenchimento deste campos :

Campos obrigatórios

O campos Cliente e Equipamento são obrigatórios, ou seja, o sistema não aceitará que estes campos estejam vazios, do contrário o mesmo mostrará uma mensagem de erro

Campos facultativos

Os campos Data de entrada, Condição de pagamento , Garantia e Prazo estimado não são estritamente obrigatórios de modo que se forem deixados em branco o que será mostrado no orçamento serão estes caracteres : #### . De todo modo, essas informações também são importantes para que o orçamento seja claro e preciso para o cliente.

Peças e serviços

Basicamente o conjunto de itens mais importantes em um orçamento, e neste sistema esta dividido da seguinte forma : Peças que serão ou que foram aplicadas e Serviços que serão ou foram executados.

Neste sistema tanto as peças quanto os serviços são adicionados em forma de itens pelo sistema, no qual esses mesmos itens precisam ter Quantidade , Descrição e Valor Unitário

Quantidade

Seja para adicionar uma peça ou um serviço é estritamente necessário descrever a quantidade de peças ou de serviços que estão sendo ou foram aplicados. O formato aceito pelo campo de Quantidade são somente números inteiros.

Descrição

É essencial que a descrição da peça ou do serviço seja coerente e objetiva para que não exista nenhum tipo de ambiguidade quanto a natureza da peça ou do serviço. Em especial no caso das peças, em alguns caso é interessante especificar informações como Marca , Código e etc.

Valor unitário

No campo Valor Unitário o sistema irá por um dado em formato monetário, ou seja, o sistema irá aceitar somente o formato com números e virgula. O Valor Total é calculado a partir da multiplação da quantidade pelo valor unitário

Se todos os dados estão corretos ao clicar no botão "Adicionar item" o sistema adicionará as informações sejam de peças aplicadas ou de serviços executados, entretanto se algum dado esitver faltando o sistema não irá prosseguir devido a falta de informações e será apresentada uma mensagem de erro.

Controle de itens

Após o botão "Adicionar item" ser clicado , sejam para Peças ou para Serviços, o sistema irá adiocionar essa informações em um item, onde o mesmo ficará a mostra no orçamento.

Dentro de cada item irão existir dois botões de controle que servem para o manuseio e edição de cada item. A função de cada um desses botões está especificada ao lado.
