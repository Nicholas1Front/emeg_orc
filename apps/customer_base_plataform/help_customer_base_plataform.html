Aqui você encontrará um guia completo sobre o funcionamento do sistema de controle de clientes e aprenderá a utilizá-lo da maneira correta.

O que é ? Como se usa ?

O sistema de controle clientes EMEG auxilia no controle de dados e informações de nossos clientes e seus respectivos equipamentos. Este sistema funciona localmente em seu computador e precisa de conexão de internet para acessar os dados e arquivos da nuvem. Importante destacar que esta plataforma web atua juntamente com o sistema de criação de orçamentos, dessa forma é estritamente necessário atenção e cuidado com o manuseio dos dados e informações.

Este sistema foi desenvolvido para fazer quatro ações relacionadas ao manuseio e manipulação dos dados:

Adicionar

Com os botões "Adicionar cliente" e "Adicionar equipamento" você poderá implementar novos dados no sistema

Excluir

"Excluir cliente" e "Excluir equipamento" darão a você a capacidade de deletar dados existentes no sistema

Editar

Para editar informações já existentes no programa, você poderá usar os botões "Editar cliente" e "Editar equipamento"

Enviar

Após todas as alterações executadas, "Enviar ao servidor" deve ser clicado para enviar os dados e informações para o mesmo

Passo a passo básico

Para que se atinja o resultado esperado, é necessário que o sistema seja usado e manipulado de acordo com a forma que foi projetado e arquitetado. Para que não existam dúvidas ou ambiguidades quanto a forma de utilizar, aqui está um passo a passo básico de como utilizar o sistema corretamente.

Abra o programa

Antes de abrir o programa para qualquer alteração, é necessário a verificação e validação do que realmente precisa-se fazer. Por exemplo : se existe a necessidade de Adicionar um equipamento se faz necessário verificar as informações do mesmo para este seja adiocionado e cadastrado de maneira correta, o mesmo principio vale para as outras necessidades como Exlcuir ou Editar.

Faça as ações necessárias

Com os objetivos devidamente explanados poderemos partir para a manipulação dos dados e informações, porém é importante destacar que as ações devem ser executadas com atenção e de acordo com o especificado, para que não hajam dados e informações incorretas no sistema.

Envie os dados para o servidor

Após toda a manipulação dos dados e informações o botão "Enviar informações ao servidor" deve ser clicado para que as alterações sejam implementadas pelo servidor na nuvem.

Aguarde o tempo de carregamento

Após a confirmação por senha, o sistema irá demorar um certo tempo para internalizar os dados e informações na nuvem, por isso enquanto o sistema estiver enviando e internalizando as informações A PÁGINA NÃO PODE SER FECHADA. Após o termino do processo a página pode ser fechada com segurança

Confirmação por senha

Assim que uma ação é executada tal como : "Adicionar", "Editar", "Excluir" ou "Enviar ao servidor", o sistema pedirá uma confirmação com senha para que você prossiga a ação desejada com segurança, preservando os dados de alterações desnecessárias ou acidentais.

Lembre-se que este sistema serve para controle e manuseio de dados e informações dos clientes, então toda ação e alteração nos dados ficará sob a responsabilidade do usuário que está usufruindo deste programa.

Mensagens do sistema

É um fato que erros são possíveis e podem acontecer em qualquer momento e seus motivos podem ser diversos : Preenchimento errado de informações, falta de dados, erros por parte do servidor ou por não se seguir os passos corretamente, enfim os erros podem ser diversos por isso este sistema conta com menasgens de erro que indicam qual é a natureza e causa do erro.Ou seja em caso de erro, análise atentamente o conteúdo da mensagem de erro para que o problema possa ser corrigido e os dados possam seguir o fluxo programado

Para fins de segurança, visando com que o fluxo do programa siga normalmente conforme programado, este sistema também conta com mensagens de sucesso para informar ao usuário que ações feitas pelo sistema foram concluidas com êxito sem nenhum tipo de problema.
