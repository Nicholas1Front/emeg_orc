Como funciona e como se usa ?

O sistema de controle clientes EMEG auxilia no controle de dados e informações de nossos clientes e seus respectivos equipamentos. Este sistema funciona localmente em seu computador e precisa de conexão de internet para acessar os dados e arquivos da nuvem. Importante destacar que esta plataforma web atua juntamente com o sistema de criação de orçamentos, dessa forma é estritamente necessário atenção e cuidado com o manuseio dos dados e informações.

Este sistema foi desenvolvido para fazer quatro ações relacionadas ao manuseio e manipulação dos dados:

Adicionar

Com os botões "Adicionar cliente" e "Adicionar equipamento" você poderá implementar novos dados no sistema

Excluir

"Excluir cliente" e "Excluir equipamento" darão a você a capacidade de deletar dados existentes no sistema

Editar

Para editar informações já existentes no programa, você poderá usar os botões "Editar cliente" e "Editar equipamento"

Enviar

Após todas as alterações executadas, "Enviar ao servidor" deve ser clicado para enviar os dados e informações para o mesmo

Assim que uma ação é executada tal como : "Adicionar", "Editar", "Excluir" ou "Enviar ao servidor", este sistema pedirá uma confirmação com senha para que você prossiga com segurança
